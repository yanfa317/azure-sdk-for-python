Back up a key in a protected form that can’t be used outside Azure Key Vault. This is intended to allow copying a key from one vault to another. Requires the key/backup permission.

Backup / restore cannot be performed across geopolitical boundaries. For example, a backup from a vault in a USA region cannot be restored to a vault in an EU region.

Parameters

Returns

async restore_key(backup: bytes, **kwargs: **Any) → azure.keyvault.keys.models.Key[source]¶

Restore a key backup to the vault. This imports all versions of the key, with its name, attributes, and access control policies. Requires the keys/restore permission.

If the backed up key’s name is already in use in the target vault, restoring it will fail. Also, the target vault must be owned by the same Microsoft Azure subscription as the source vault.

Parameters

backup (bytes) – The raw bytes of the key backup

Returns

The restored key

Return type

Key

Raises
